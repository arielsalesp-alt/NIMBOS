# טופס הזמנה נימבוס

פרויקט סטטי לטופס הזמנה ללקוחות נימבוס הוט מול 99 דיגיטל.

## שימוש

1. פותחים את `index.html` בדפדפן.
2. ממלאים את פרטי ההזמנה.
3. לוחצים על `שמירת טיוטה` כדי לשמור בדפדפן.
4. לוחצים על `יצירת PDF לשליחה`.
5. בחלון ההדפסה בוחרים `Save as PDF` או `Microsoft Print to PDF`.

הטיוטה נשמרת מקומית בדפדפן בלבד.
